# Blackjack (21) Game – Python & Tkinter

This project is a **graphical Blackjack (21) game** developed using **Python** and **Tkinter**.  
It simulates the classic casino Blackjack experience with betting mechanics, dealer AI, and a user-friendly interface.

## 🎰 Features
- 🃏 Classic Blackjack (21) rules
- 💰 Starting balance system
- 🎲 Betting system with balance control
- 🏦 Dealer (casino) AI logic
- 🂠 Hidden dealer card mechanic
- 🔢 Automatic score calculation (Ace 1 / 11 logic)
- 🎉 Blackjack bonus (1.5x payout)
- 🤝 Draw (push) handling
- 🔄 New game / reset functionality
- 🎨 Graphical interface with card symbols and colors

## 🛠 Technologies Used
- Python 3
- Tkinter (GUI)
- `random` module (deck shuffling)
- Object-Oriented Programming (OOP)

## ▶️ Run the Program
```bash
python blackjack.py
```
